ServiceJour

Description

Le fichier batch `serviceJour.bat` offre une interface simple en ligne de commande permettant à l'utilisateur de choisir entre afficher l'heure du moment, afficher la date du moment, ou quitter le programme. Il peut effectuer ces actions de manière indépendante, permettant à l'utilisateur de choisir de ne pas afficher l'heure et d'avoir la date, ou de tout afficher.

Option du menu :

  • L'option 1 affiche l'heure actuelle. Le programme utilise la commande `time /t` pour afficher l'heure, met en pause le script pour permettre à l'utilisateur de voir l'heure puis retourne au menu principal.
  • l'option 2 permet d'afficher la date actuelle. le porgramme utilise la commande `date /t` pour afficher la date puis pause le script pour permettre à l'utilisateur de voir la date et retourne au menu principal.
  • l'option 3 permet de quitter le programme en Affichant un message de départ. Cela utilise la commande `exit /b 0` pour arreter le script.

Utilisation

  • 1. Double-cliquez sur le fichier `serviceJour.bat` pour l'exécuter.
  • 2. Choisissez l'option correspondante en entrant le numéro associé.
  • 3. Suivez les instructions à l'écran.
  • 4. Répétez le processus ou choisissez l'option pour quitter.

Remarques

  • Les commandes `time /t` et `date /t` sont utilisées pour afficher l'heure et la date respectivement.
  • Le script utilise `set /p` pour permettre à l'utilisateur de saisir son choix.
  • En cas de choix invalide, le script affiche un message et retourne au menu principal
